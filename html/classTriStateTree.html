Detailed Description

The TriStateTree class helps facilitate the efficient evaluation of queries.

The tree is built from a query with each leaf node representing a search, all other nodes representing the search operators (and, or etc). Each node can be one of three states: Unknown, False, or True. As the result of a search for the a game is discovered, the state for the corresponding leaf node is set to true or false. After each update the tree is re-evaluated, returning a value of Uknown, False, or True. In some cases the value of the tree can be determined before all search results are known. Eg. if the query is A or B, then if A is true then we know the result is true without having to evaluate B.
