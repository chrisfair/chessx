Detailed Description

TagValues class holds a list of integer values.

The list represents all the values of a particular tag in the database. For integer values, the actual values are not stored. The reasoning is that you have to remember the index anyway, which is equivalent to remembering the actual value. For this reason request for a value at a particular index, actually returns the index.
