Detailed Description

The FilterSearch class is not a real search class.

It is used to combine other searches with a filter, in order to solve the search tree quicker if possible. For instance if a search is combined with a filter using the AND operator, only games in the filter are searched. If it is combined with the OR operator, only games not in the filter are searched.
