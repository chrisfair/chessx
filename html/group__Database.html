§ IndexBaseType

typedef qint64 IndexBaseType

The PgnDatabase class provides database access to PGN files.

The class is derived from the Database class, providing methods for the loading and saving of games, and for performing searches and queries.

§ TagIndex

typedef quint32 TagIndex

The IndexItem class holds one item that is listed in a index It contains essential game header information that is kept in memory for fast access.

It is used in conjunction with the Tags class. For most items it stores the id that refers to the relevant tag item in the Tags instance.
