Detailed Description

Store and combine simple and complicated queries.

This is done by using reverse polish (postfix) notation. The Query class is a conceptual list of operands (different kinds of searches) and operators (logical AND, OR, NOT), and methods to manipulate this list. Example (A && B) || (C && D) where A,B,C,D are different searches and &&=And, ||=Or would be stored as a list: A B && C D && ||
